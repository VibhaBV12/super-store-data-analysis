# super-store-data-analysis
# 📊 Superstore Sales Data Analysis using Pandas

## 📌 Project Overview

This project focuses on exploring and understanding the **Sample Superstore** dataset using the **Pandas** library in Python. The analysis was performed in **Jupyter Notebook (Anaconda)** to become familiar with basic data exploration techniques and understand the structure of a real-world dataset before performing detailed analysis.

---

## 🛠️ Technologies Used

* Python
* Pandas
* Jupyter Notebook (Anaconda)

---

## 📂 Dataset

The dataset used in this project is the **Sample Superstore** dataset, which contains information about customer orders, sales, profits, discounts, shipping details, products, and regions.

---

## 📖 Pandas Operations Performed

The following Pandas functions and attributes were explored during this project:

* `head()` – Display the first five rows of the dataset.
* `tail()` – Display the last five rows.
* `sample()` – View random records from the dataset.
* `info()` – Check column names, data types, and missing values.
* `describe()` – Generate statistical summaries of numerical columns.
* `sort_values()` – Sort the dataset based on selected columns.
* `index` – View the row index.
* `dtypes` – Display the data type of each column.
* `axes` – Display the row and column labels.
* `shape` – Find the number of rows and columns.
* `size` – Calculate the total number of elements.
* `ndim` – Determine the number of dimensions.
* `memory_usage()` – Check memory consumption of the dataset.
* `len()` – Find the total number of rows.
